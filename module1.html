Module 1: HTML Basics

1. HTML Headings

We have 1-6 headings ranging from the most important text (largest text heading size) to the least important text (smallest text heading size).

  • <h1> defines the most important heading.
  • <h6> defines the least important heading.

Example Code:

bash — 80×24 
<h1>This is heading 1</h1>
<h2>This is heading 2</h2>
<h3>This is heading 3</h3>

2. HTML Paragraphs

Paragraphs are defined with the <p> tag.

Example Code:

bash — 80×24 

<p>This is a paragraph.</p>
<p>This is another paragraph.</p>

Combined Example (Full Structure):

bash — 80×24 

<!DOCTYPE html>
<html>

<head>
<title> My Website </title>
</head>

<body>

<h1>Welcome to my homepage!</h1>
<p>My first website</p>

</body>
</html>

3. HTML Images

HTML images are defined with the <img> tag. Unlike other HTML tags, it has no ending tags like </img>.

Attributes:

  • src: Specifies the path to the image file.
  • alt: Alternative text used to describe the image if it doesn't load.
  • width/height: Used to input the size (number or percentage) to manipulate the image's size.

Example Code:

bash — 80×24 
<img src="flower.jpg" alt="Pink tulips" width="104" height="142">

Note: The image file must be .jpg for this specific example code to work successfully.

4. HTML Videos

The <video> tag is used to embed video content in a document, such as a movie clip or other video streams. It has one or more source tags with different video sources, but the browser will choose the first source it supports.

Supported Formats: MP4, WebM, and OGG.

Example Code:

bash — 80×24 
<video width="320" height="240" controls>
  <source src="assets/EDGE_OF_INPHINITY.mp" type="video/mp4">
  <source src="movie.ogg" type="video/ogg">
  Your browser does not support the video tag.
</video>

The text between the video tags will only be displayed in browsers that do not support the video element.

Attributes:

  • src: specifies the URL of the video file
  • width: sets the width size of the video
  • muted: mutes the audio of the video
  • loop: starts the video over again once it is finished
  • autoplay: automatically starts playing the video as soon as the webpage has loaded or when the video is ready

5. HTML Audios

The <audio> tag is used to embed sound content in a document, such as music or other audio streams. Just like the video tag, it also has one or more source tags with different audio sources, and the browser chooses the first source it supports.

Supported Formats: MP3, WAV, and OGG.

Example Code:

bash — 80×24 
<audio controls>
  <source src="horse.ogg" type="audio/ogg">
  <source src="horse.mp3" type="audio/mpeg">
  Your browser does not support the audio tag.
</audio>

Attributes:

  • src: specifies the URL of the audio file
  • muted: mutes the audio
  • loop: starts the audio over again once it is finished
  • autoplay: automatically starts playing as soon as the webpage has loaded

6. HTML Links

HTML links are hyperlinks defined with the <a> tag (anchor tag) and href (Hypertext Reference) for linking other webpages to a document.

Example Code:

bash — 80×24 
<a href="https://www.google.com">This is a link</a>
  • The link's destination is specified in the href attribute.
  • Attributes are generally used to provide additional information about HTML elements.

7. HTML Forms

A section of a document that contains normal content, markup, special elements called controls (checkboxes, radio buttons, menu, etc.) and labels to said controls.

The <input> Element

This is the most used form element. The "type" attribute specifies the type of input element to display.

Common Attributes:

  • Name: Each input field must have a name attribute to be submitted.
  • Value: Specifies an initial value for an input field.
  • Placeholder: Specifies a short hint of what the expected value is to be inputted in the form.
  • Required: Specifies that an input must be filled out before submitting the form.

Input Element Types & Examples:

Advance Lesson: External CSS (Preview for Module 2)

This is most recommended when you will be full-on using CSS for designing your website. CSS parts are done on a separate file that you will only need to link to your HTML file in order to display the input of the CSS code!

External styles are defined within the <link> element, inside the <head> section of an HTML page.

HTML Example:

bash — 80×24 
<!DOCTYPE html>
<html>
<head>
<link rel="stylesheet" href="mystyle.css">
</head>
<body>

<h1>This is a heading</h1>
<p>This is a paragraph.</p>

</body>
</html>

CSS File Example (mystyle.css):

bash — 80×24 
body {
  background-color: lightblue;
}

h1 {
  color: navy;
  margin-left: 20px;
}

Key Notes:

  • The file name in the href attribute is labeled .css, which defines that file as only for CSS codes!
  • Both internal and external CSS are located in the <head> tag of the HTML file.
  • Stylesheet is the standard link to be used in order for the HTML and external CSS to be connected.
