Module 3: JavaScript Fundamentals

What is JavaScript?

JavaScript is a lightweight, interpreted programming language commonly used for creating dynamic and interactive web elements. It runs directly in the browser — no installation needed.

1. JavaScript Syntax

There are 3 ways to incorporate JavaScript into your code: Inline, Internal, and External JS.

External and Internal JS require the <script> tag, which alerts the browser to interpret the content inside it.

Internal JavaScript

Internal JS is placed within the <body> or <head> of the HTML.

Note: The last command will always be the one applied!

bash — 80×24 
<script>
  document.getElementById("demo").style.color = "blue";
</script>

1. JavaScript Syntax — continued

External JavaScript

The external JS file has the extension .js and must be in the same folder (or a correct path). It is linked using a <script src=""> tag.

External JS is written the same way as Internal JS — just without the <script> tag wrapping it.

bash — 80×24 
<!-- In your HTML -->
<script src="myscript.js"></script>

/* Inside myscript.js */
document.getElementById("demo").style.color = "blue";

Inline JavaScript

Inline JS requires an existing HTML tag and must be placed as an attribute within that tag. For buttons, the onClick attribute is used.

bash — 80×24 
<button onClick="alert('This is an Alert!')">Click Me!</button>

2. Document Object Reference

Method

A method is an action — such as adding, replacing, or deleting an HTML element.

Property

A property is a value that can be changed or set by code. Methods act on properties.

document.getElementById()

document.getElementById() targets a specific element with a unique ID to modify or change it. Note: it is case-sensitive — the E, B, and I are always capitalized.

For Non-INPUT Elements — use .innerHTML

bash — 80×24 
document.getElementById('p1').innerHTML = "Success!";

For INPUT Elements — use .value

bash — 80×24 
document.getElementById('input1').value = "Success!";

3. Variables

Variables store information or data — like boxes that hold objects. However, a variable can only hold one piece of information at a time.

Declaring Variables

  • let — the most common, modern way to declare a variable.
  • var — the old-school way; functionally similar to let.
  • const — short for constant; the value cannot be changed after assignment.

Multiple Variable Declaration:

bash — 80×24 
let user = "John", age = 25, message = "Hello";

You CANNOT redeclare the same variable:

bash — 80×24 
let message = "hi!";
let message = "hello!"; // ❌ SyntaxError!

Naming Rules

  1. Digits cannot be the first character.
  2. Variables must only contain letters, digits, $, and _.

Variable names are case-sensitive: LastName ≠ lastname.

Data Types

  • String — pure text, placed within quotation marks ("" or '').
  • Integer — whole digits/numbers, not in quotation marks.
  • Float — numbers in decimal form.

4. Math / Arithmetic Operators

Arithmetic operators perform calculations on numbers.

  • + Addition — adds values together.
  • - Subtraction — subtracts the second value from the first.
  • * Multiplication — multiplies values.
  • / Division — divides values.
  • % Modulo — returns the remainder after division.
bash — 80×24 
15 + 3  → 18      // Addition
15 - 3  → 12      // Subtraction
15 * 3  → 45      // Multiplication
15 / 3  → 5       // Division
10 % 3  → 1       // Modulo (remainder)

5. Assignment Operators

Assignment operators assign values to variables.

bash — 80×24 
let x = 10;

x = 10     // Assign       → x is 10
x += 5     // Add & Assign → x = 10 + 5  = 15
x -= 3     // Sub & Assign → x = 10 - 3  = 7
x *= 2     // Mul & Assign → x = 10 * 2  = 20
x /= 4     // Div & Assign → x = 10 / 4  = 2.5

Comparison Operators & Boolean

A Boolean can only be true or false. Comparison operators compare two values and return a Boolean result.

bash — 80×24 
let x = 10, y = 5;

x == y   // equal to           → false
x != y   // not equal to       → true
x > y    // greater than       → true
x < y    // less than          → false
x >= y   // greater or equal   → true
x <= y   // less or equal      → false

6. Functions

A function is a block of code designed to perform a particular task. It is executed when called (invoked).

Function Declaration

Use the keyword function, followed by the function name, parentheses (), and curly braces {} containing the code to execute.

bash — 80×24 
function greet() {
    document.getElementById("greeting").innerHTML = "Hello, World!";
}

Function Invocation

To execute the function, call its name followed by parentheses — or trigger it via an HTML event.

bash — 80×24 
greet(); // invokes the function in a script

<button onClick="greet()">Click Me to Greet!</button>

7. Conditionals

Conditionals let your code make decisions. They check whether a condition is true or false and run different blocks of code accordingly.

if

The if block runs only when its condition is true. If the condition is false, the block is skipped entirely.

bash — 80×24 
let score = 80;

if (score >= 75) {
    console.log("You passed!");
}

else if

else if provides an additional condition to check when the previous if (or else if) was false. You can chain as many as you need.

bash — 80×24 
let score = 65;

if (score >= 90) {
    console.log("Excellent!");
} else if (score >= 75) {
    console.log("You passed!");
} else if (score >= 60) {
    console.log("Barely passed.");
}

else

else is a catch-all — it runs when none of the preceding conditions were true. It has no condition of its own.

bash — 80×24 
let score = 45;

if (score >= 90) {
    console.log("Excellent!");
} else if (score >= 75) {
    console.log("You passed!");
} else if (score >= 60) {
    console.log("Barely passed.");
} else {
    console.log("You failed. Try again!");
}
