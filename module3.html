Module 3: JavaScript Fundamentals

What is JavaScript?

JavaScript is a lightweight interpreted programming language commonly used for creating dynamic and interactive web elements.

1. JavaScript Syntax

There are 3 ways to incorporate JavaScript into your code: Inline, External, and Internal JS!

External and Internal JS require the <script> tag. The <script> tag alerts the browser to interpret the content in between.

Internal JavaScript

Internal JS is placed within the body or the head of the HTML.

Note: The LAST command will ALWAYS be the one that will be applied!

Format of Internal JS:

bash — 80×24 
<script>

    [pointer] {
        [attribute]: [value];
        [attribute]: [value];
        [attribute]: [value];
    }

</script>

Example of Internal JS:

bash — 80×24 
<script>

    p {
        font-size: 12px;
        color: black;
        text-decoration: wavy;
    }

</script>

External JavaScript

The external JS has the file extension ".js" in order to work and must also be placed in the same folder!

External JS is the same as internal JS—just without the <script> tag.

Format of External JS:

bash — 80×24 
[pointer] {
    [attribute]: [value];
    [attribute]: [value];
    [attribute]: [value];
}

Example of External JS:

bash — 80×24 
p {
    font-size: 12px;
    color: black;
    text-decoration: wavy;
}

Inline JavaScript

Inline JS—just like in CSS—requires an existing tag and needs to be WITHIN the tag.

Inline JS requires the "onClick=" attribute for buttons.

Example of Inline JS:

bash — 80×24 
<button onClick="alert('This is an Alert!')">Click Me!</button>

2. Document Object Reference

Method

What is a Method? A method is an action, such as: Adding, Replacing, or Deleting an HTML element.

Property

Property is a value that can be changed or set according to the written code. They are what the METHODS are used on.

document.getElementById()

document.getElementById() is used for targeting a specific element with a unique ID in order to modify or change it.

Note: document.getElementById() is case-sensitive. This means that 'E', 'B', and 'I' are ALWAYS capitals and everything else is lowercase.

Whatever comes after this determines what will happen to the code that has been pointed to. However, before the change is applied, specifying what type of element it is comes first. There are two types of these 'identifiers':

For Non-INPUT Elements:

These are elements that are not <input>.

Examples of non-input elements: <p>, <h1>, <h4>, etc.

For non-INPUT elements, we use ".innerHTML".

Example:

bash — 80×24 
document.getElementById('p1').innerHTML = "success!";

For INPUT Elements:

For INPUT elements, we use ".value".

Example:

bash — 80×24 
document.getElementById('input').value = "success!";

3. Variables

Variables in JS are what you use to store information or data.

  • They are basically like boxes; they store different "objects".
  • However, unlike boxes, variables CANNOT store different information at once—they can only store one piece of information at a time.

Declaring Variables

In JS, to "declare" a variable, we must use the following:

  • "let" - This is the most common and modern way of declaring a variable.
  • "var" - This is used in the old-school way of declaring a variable. This has no difference with "let".
  • "const" - Short for constant. Unlike the aforementioned two, this is used when the information needs to be irreplaceable and CANNOT change.

Multiple Variable Declaration:

You can declare multiple variables at once!

bash — 80×24 
let user = "John", age = 25, message = "Hello";

What is a Declaration?

"Declaration" is what you call the entirety of where "let" or "var" declares a variable.

bash — 80×24 
let message = "hi!";

This entire string of code is what you call a declaration!

Important Note:

While you can declare multiple variables at once, you cannot, however, declare a variable multiple times!

bash — 80×24 
let message = "hi!";
let message = "hello!"; // This leads to a Syntax Error!

Naming Variables

In naming variables, there are 2 absolute rules:

  1. Digits CANNOT be the first character of the variable.
  2. Variables must ONLY contain letters, digits, and the symbols "$" and "_".

Examples of WRONG variable names:

  • my-name
  • 123abc
  • 1name
  • first-name
  • last-name

Note: In naming variables, you have to note that variables are CASE-SENSITIVE. This means that LastName and lastname are NOT the same and WILL lead to an error.

Capitalization Methods:

There are two ways of capitalizing variable names that are a combination of different words:

  • Camel Casing - This is the type of casing that capitalizes the first letter of the words that are included in the name.
    Examples: colorOrange, redFlower, etc.
  • Snake Casing - This is utilized by using "_" in order to create space.
    Examples: color_orange, red_flower, etc.

Data Types

There are 3 different data types:

  • String - It is the type of data that is purely text and is placed within quotation marks ("" or '').
  • Integer - It is the type of data that consists of digits/numbers and is not placed within quotation marks.
  • Float - It is the type of data that also consists of numbers but in decimal form.

4. Math/Arithmetic Operators

Arithmetic Operators perform arithmetic on numbers!

Addition

With the symbol plus (+), it is used to add the values together to get the resulting number.

bash — 80×24 
15 + 3 would result in 18
3 + 5 would result in 8
200 + 20 would result in 220
40 + 2 would result in 42

Subtraction

With the symbol hyphen (-), it is used when subtracting the second value from the first value.

bash — 80×24 
15 - 3 would result in 12
3 - 5 would result in -2
200 - 20 would result in 180
40 - 2 would result in 38

Multiplication

With the symbol asterisk (*), it is used when multiplying values.

bash — 80×24 
15 * 3 would result in 45
3 * 5 would result in 15
20 * 10 * 2 would result in 400
40 * 2 would result in 80

Division

With the symbol forward slash (/), it is used when dividing values.

bash — 80×24 
15 / 3 would result in 5
10 / 2 would result in 5
200 / 20 would result in 10
40 / 2 would result in 20

Modulo

With the symbol percent (%), it is used when finding the remainder after division. It is basically a modified version of division; instead of finding the quotient, it finds the remainder.

bash — 80×24 
10 % 3 would result in 1
35 % 6 would result in 5
15 % 2 would result in 1
44 % 5 would result in 4

5. Assignment Operators

Assignment Operators are used to assign values to variables.

Assign Value

With the symbol (=), it is the most straightforward way to assign a value to a variable as it directly assigns the variable whatever is written right after.

bash — 80×24 
x = 10 is equivalent to x = 10
y = 5 is equivalent to y = 5
z = 2 is equivalent to z = 2

Add & Assign

With the symbol (+=), it is used when adding a value to the declared value of the variable before assigning it.

bash — 80×24 
let x = 10;

x += 5 is equivalent to x = 10 + 5, assigned as x = 15
x += 10 is equivalent to x = 10 + 10, assigned as x = 20
x += 50 is equivalent to x = 10 + 50, assigned as x = 60

Subtract & Assign

With the symbol (-=), it is the subtraction version of Add & Assign, as it is used when subtracting a value from the declared value of the variable before assigning it.

bash — 80×24 
let x = 50;

x -= 5 is equivalent to x = 50 - 5, assigned as x = 45
x -= 10 is equivalent to x = 50 - 10, assigned as x = 40
x -= 50 is equivalent to x = 50 - 50, assigned as x = 0

Multiply & Assign

With the symbol (*=), it is used when multiplying a value to the declared value of the variable.

bash — 80×24 
let x = 3;

x *= 5 is equivalent to x = 3 * 5, assigned as x = 15
x *= 3 is equivalent to x = 3 * 3, assigned as x = 9
x *= 30 is equivalent to x = 3 * 30, assigned as x = 90

Divide & Assign

With the symbol (/=), it is used when dividing a value to the declared value of the variable.

bash — 80×24 
let x = 100;

x /= 10 is equivalent to x = 100 / 10, assigned as x = 10
x /= 5 is equivalent to x = 100 / 5, assigned as x = 20
x /= 2 is equivalent to x = 100 / 2, assigned as x = 50

Boolean Data Type

Boolean is a data type that can only have two values: true or false.

Comparison Operators

Comparison operators are used to compare two values. They return true or false.

bash — 80×24 
let x = 10;
let y = 5;

x == y is equivalent to x = y, output is false
x != y is equivalent to x ≠ y, output is true
x > y is equivalent to x > y, output is true
x < y is equivalent to x < y, output is false
x >= y is equivalent to x ≥ y, output is true
x <= y is equivalent to x ≤ y, output is false


6. Functions

A function is a block of code designed to perform a particular task. It is executed when "called" or "invoked".

Function Declaration

To declare a function, we use the keyword function, followed by the name of the function, parentheses (), and curly braces {} that contain the code to be executed.

Example of Function Declaration:

bash — 80×24 
function greet() {
    document.getElementById("greeting").innerHTML = "Hello, World!";
}

Function Invocation

To execute the function, we simply call its name followed by parentheses.

bash — 80×24 
greet(); // Invokes the greet function
<button onClick="greet()">Click Me to Greet!</button>
