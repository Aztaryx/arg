Module 2: CSS Fundamentals

What is CSS?

CSS (Cascading Style Sheets) is the language used to style HTML content on the web. It tells browsers how to display elements on screen, on paper, or in other media. Because styles can be defined in external stylesheet files, the same rules can be applied across multiple pages, which saves time and keeps design consistent.

  • CSS = Cascading Style Sheets
  • Describes how HTML elements are displayed
  • External stylesheets (files with .css) can control many pages at once

CSS Syntax

A CSS rule has a selector and a declaration block. The selector targets elements you want to style. Declarations are property:value pairs, separated by semicolons, inside curly braces.

bash — 80×24 
p {                 
  color: red;        
  text-align: center;
}

Explanation:

  • p — selector targeting <p> elements.
  • color: red — sets text color.
  • text-align: center — centers inline content.

The CSS Box Model

Every HTML element is treated as a rectangular box consisting of, from inside out: content, padding, border, and margin. These parts determine element size and spacing.

  • Content — where text and images appear.
  • Padding — transparent space around content.
  • Border — border around padding and content.
  • Margin — transparent space outside the border.

Border Styles

The border-style property controls how a border looks. Common values:

  • dotted — dotted border
  • dashed — dashed border
  • solid — solid border
  • double — double border
  • groove, ridge, inset, outset — 3D effects depending on border-color
  • none, hidden — no visible border

You can specify 1–4 values to set top/right/bottom/left border styles independently.

Pseudo-classes

Pseudo-classes (like :hover) are appended to selectors to target elements in a specific state (for example when a user hovers, focuses, or visits a link).

Common link pseudo-classes:

  • :link — unvisited links
  • :visited — visited links
  • :hover — when the mouse is over an element
  • :active — while an element is being activated
